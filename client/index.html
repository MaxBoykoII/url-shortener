Basejump: URL Shortener Microservice

  • Given a url that follows the http://www.example.com format, this microservice returns a JSON object that includes a shortened url. Visiting the shortened version redirects the user to the original url.
  • Given an invalid url, the microservice returns a JSON object with an error message.

Example Creation Usage

https://url-shortener-2.herokuapp.com/new/https://www.google.com
https://url-shortener-2.herokuapp.com/new/http://www.cnn.com

Example Output

{"original_url":"https://www.google.com","short_url":"https://url-shortener-2.herokuapp.com/41Dr_leOl"}
{"original_url":"http://www.cnn.com","short_url":"https://url-shortener-2.herokuapp.com/E1ohkl-_x"}

Example Usage

The shortened url https://url-shortener-2.herokuapp.com/E1ohkl-_x, for example, redirects to http://www.cnn.com.
