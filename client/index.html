Basejump: URL Shortener Microservice

Given a url that follows the http://www.example.com format, this microservice returns a JSON object that includes a shortened url. Visiting the shortened version redirects the user to the original url. Given an invalid url, the microservice returns a JSON object with an error message.

Example Usage

https://timestamp-microservice-2.herokuapp.com/March 20 2001
https://timestamp-microservice-2.herokuapp.com/March%2020%202001

Example Output

{"unix":985046400,"natural":"March 20, 2001"}
