Url Shortener

This microservice accepts a string as a parameter and checks whether it is a unix timestamp or a natural language date. If it is, the microservice returns both the unix timestamp and the natural language date. If not, the microservice returns null for both properties.

Example Usage

https://timestamp-microservice-2.herokuapp.com/March 20 2001
https://timestamp-microservice-2.herokuapp.com/March%2020%202001

Example Output

{"unix":985046400,"natural":"March 20, 2001"}
